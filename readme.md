## thinkphp5.0.23 rce

## 安装

```
git clone https://github.com/1derian/thinkphp5-rce.git
cd thinkphp5-rce
pip3 install -r requirements.txt
```

## 使用

poc

```
python3 tppoc.py -u http://www.xxx.com
```

exp

```
python3 tpexp.py -u http://www.xxx.com -c id
```
